2) GT Movies Store — Description

GT Movies Store is a Django-based web app that lets users browse a movie catalog, view details, and manage entries via an admin interface. It demonstrates routing, templates, models, and database integration.

User Stories & Screens

  • Home: Welcome/landing with quick links to catalog and about.
  • Catalog: List of movies (title, genre, year, rating) with search/filter.
  • Details: Dedicated page per movie with extended description and imagery.
  • Admin: Create/update/delete movies to keep the catalog fresh.

3) Process — How I Built It

  • Planning & Setup: New Django project; clean app structure.
  • Implementation: URLs → views → templates; added static files for styling.
  • Database: MySQL + PyMySQL; migrations for models.
  • Testing/Debugging: Solved config and import issues; verified routes and templates with unit runs.
  • Deployment: Source on GitHub; deployed on PythonAnywhere.
